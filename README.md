# SplitShare

SplitShare is a program that splits the expenses among friends
This program written in C++ in three files namely server.h, sever2.cpp and client.cpp
To use this run the SplitShare.exe
Then add the details of all the members
Then add their expenses and then add the number of people you want to divide those expenses into
You can then calculate total expenses, split the expenses, see all expenses, see all pending balances and you can also settle up balances which shows who owes whom how much money
