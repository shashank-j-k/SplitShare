Expense Splitter (C++ Console Application)
Overview

Expense Splitter is a C++ console-based application designed to manage and settle shared expenses among a group of people. The project allows users to add members, record expenses, split expenses either equally or among selected members, track pending balances, and finally settle all dues with minimal transactions.

This project was built to demonstrate core C++ concepts, including:

Object-Oriented Programming (OOP)

Dynamic memory management

Doubly linked lists

Pointer manipulation

Basic algorithmic problem solving
